# 💿Кастомные диски

{% hint style="warning" %}
Для работы радио необходим мод **Simple Voice Chat** на клиенте.
{% endhint %}

### ✍️ **Команды**

```
/cd download "Прямая ссылка" "Название файла" - Загрузить свою музыку на сервер
/cd create Название файла Название диска - Создать музыкальный диск
```

#### 🎵 **Поддерживаемые форматы аудиофайлов**

🔹 `MP3`\
🔹 `WAV`\
🔹 `FLAC` (lossless)

**📂 Название файла:**\
⚠️ Указывайте **точное название загружаемого файла** (с расширением, например: `my_track.mp3`), а не произвольное имя.

**💿 Название диска:**\
✨ Может быть любым на ваше усмотрение.

**🔗 Как получить прямую ссылку на скачивание:**

1. Загрузите файл в **Google Drive**.
2. Откройте доступ **«по ссылке»** (настройки доступа — «Все, у кого есть ссылка»).
3. Скопируйте ссылку и вставьте в сервис:\
   → [https://lonedev6.github.io/gddl/](https://lonedev6.github.io/gddl/)
4. Полученную **прямую ссылку** используйте для загрузки.

**💡 Примечание:**\
Не забывайте заключать названия в **кавычки**, как указано в командах!
