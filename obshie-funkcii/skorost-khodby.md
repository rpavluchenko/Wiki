# 🚶 Скорость ходьбы

Позволяет настроить скорость передвижения под себя.

***

### Использование

```
/walkspeed <скорость>
```

| Значение | Скорость        |
| -------- | --------------- |
| `1`      | Стандартная     |
| `0.8`    | Чуть медленнее  |
| `0.5`    | Средняя         |
| `0.3`    | Медленная       |
| `0.1`    | Очень медленная |
