# 🖼️ Редактирование рамок

***

### Как открыть меню

{% hint style="info" %}
Рамка должна быть **пустой** или **зафиксированной**.
{% endhint %}

1. **Присядь** (`Shift`)
2. **Нажми `ПКМ`** по рамке

Откроется меню **«Редактирование»**.

***

### Меню настройки

| Иконка         | Название          | Описание                                         |
| -------------- | ----------------- | ------------------------------------------------ |
| 👁️ Око Эндера | **Видима**        | Рамка видна — нажми, чтобы скрыть                |
| ✖️ Барьер      | **Невидима**      | Рамка скрыта — нажми, чтобы показать             |
| 🪨 Булыжник    | **Свободна**      | Предмет можно взять — нажми, чтобы зафиксировать |
| ⬛ Бедрок       | **Зафиксирована** | Предмет нельзя взять — нажми, чтобы открепить    |

{% hint style="warning" %}
Если рамка **не пустая и не зафиксированная** — меню не откроется. Сначала забери предмет или зафиксируй рамку.
{% endhint %}
