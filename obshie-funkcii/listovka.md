# 📃 Листовка

***

### Создание

1. Возьми **бумагу** в руку
2. Нажми `ПКМ` по **кафедре**
3. Откроется меню — введи название и получи листовку (необязательно)

***

### Смена скина

Положи листовку и черепок на **наковальню** — скин листовки изменится.

| Черепок    | Скин      |
| ---------- | --------- |
| Шахтёр     | Вариант 2 |
| Руки вверх | Вариант 3 |
| Вихрь      | Вариант 4 |
| Друг       | Вариант 5 |

***

### Размещение

1. Возьми листовку в руку **(ровно 1 штуку)**
2. Нажми `ПКМ` по поверхности — листовка прикрепится к блоку

Работает на стенах, полу и потолке. Листовка автоматически ориентируется лицом к тебе.

{% hint style="warning" %}
Если блок под листовкой сломают — она выпадет как предмет.
{% endhint %}

***

### Заполнение содержимым

1. Возьми листовку в руку **(ровно 1 штуку)**
2. Нажми `ПКМ` по **кафедре**
3. Откроется меню **«Перед / Зад»** — сетка 3×3 для каждой стороны

| Предмет              | Результат                    |
| -------------------- | ---------------------------- |
| Написанная книга     | Отображает текст на листовке |
| Голова игрока        | Отображает скин              |
| Любой другой предмет | Отображается иконкой         |

{% hint style="warning" %}
Нельзя перетаскивать предметы через `Shift+ЛКМ` — только прямое перемещение в слот. Некоторые слоты принимают только определённые типы предметов.
{% endhint %}

***

### Чтение

Подойди к листовке — она подсветится оранжевым и появится подсказка.

Нажми `ПКМ` — листовка раскроется перед тобой.

* Нажми `F` (смена руки), чтобы перевернуть на обратную сторону
* Отойди дальше **3 блоков** — листовка закроется

***

### Подбор

Подойди к листовке без предмета в руке — она подсветится. **Присядь** (`Shift`) и нажми `ПКМ` — листовка вернётся в руку.
