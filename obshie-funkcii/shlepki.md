# 👋 Шлепки

Нажми `ПКМ` по другому игроку — ты хлопнешь его по плечу или шлёпнешь по Жопе. Оба игрока увидят сообщение, а все в радиусе 5 блоков услышат звук.

***

### Описание игрока

У каждого игрока есть короткое описание — оно показывается тому, кто тебя хлопнул.

| Команда             | Действие                    |
| ------------------- | --------------------------- |
| `/pame set <текст>` | Установить своё описание    |
| `/pame info`        | Посмотреть текущее описание |

{% hint style="info" %}
Если описание не задано — отображается **«Не указано»**.
{% endhint %}

***

### Ограничения

* Кулдаун между хлопками: **3 секунды**
* Приседание (`Shift`) во время ПКМ — хлопок отменяется
