# 👄 Плевки

{% hint style="warning" %}
Доступно только с привилегией **Plus**.
{% endhint %}

Плюй в направлении взгляда. Если попадёшь в игрока — он увидит уведомление.

***

### Использование

Введи команду `/spit`

* Дальность: **30 блоков**
* Кулдаун: **3 секунды**
